# CTR TS Dev Board

## Information
---
The CTR Target Board is a board that is designed to test the relations of how each component will intereact once in an assembled CTR unit.

## Versions (Hardware)
---
- Main Board
  - CTR-MOTHER-X1
  - CTR-MOTHER-X3
  - CTR-MOTHER-X4
- Power Management IC
  - CTR-TEG2(C)-PMIC-X2
- SoC
  - CTR-TS-CPU-X1
  - CTR-TEG2-CPU-X1
- Codec IC
  - CTR-TEG2-CODEC-X1
  - CTR-CODEC-X3
  - CTR-CODEC-X4
- NAND IC
  - X3
- Camera and Antenna
  - CTR-TEG2-CAM-SUB1-X1

# Nintendo 3DS

## Information
---
The Nintendo 3DS is a handheld video game console released in 2011. It goes by the codename of **CTR**, or **Citrus**

## Versions (Hardware)

    - **CTR-IR-01**
  - Main SoC

    - **CTR-CPU-40**
